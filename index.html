HTML

The follow is a list of the HTML elements listed on the Codecademy cheatsheets.

HTML Elements
Tag Name Description
<element></element> Element Content The content of an HTML element is the information between the opening and closing tags of an element.
<xmp> HTML Example Element Renders the text between the start and end tags without interpreting the HTML in between.
Note: This will create a link break.
&lt; Left Tag Symbol written within HTML to explicitly define less than symbol.
&gt; Right Tag Symbol written within HTML to explicitly define greater than symbol.
<li> List Item Element The list item element create list items inside:
  • Ordered lists <ol>
  • Unordered lists <ul>
<video> Video Element The <video> element embeds a media player for video playback. The src attribute will contain the URL to the video. Adding the controls attribute will display video controls in the media player.
